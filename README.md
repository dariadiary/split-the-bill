# Split the Bill — Simple Bill Splitting App

## Project Description
Split the Bill is a small web app that helps you quickly split a bill between people and calculate how much each person should pay.

The app is minimal, practical, and focused on clean UI and simple logic.

This project was created as part of the **JavaScript course by Canshecode**, to practice component structure, state management, and basic calculations.

## Features
- Enter the total bill amount
- Choose the number of people
- (Optional) Add tips / service fee (if implemented)
- Instant calculation of the amount per person
- Clean, responsive layout

## Technologies Used
- JavaScript
- HTML5
- CSS3
